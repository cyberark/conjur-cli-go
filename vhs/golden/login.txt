> conjur login --help

  Authenticate with Conjur using the provided identity and password.

  The command will prompt for identity and password if they are not provided via flags.

  On successful login, the password is exchanged for the user's API key, which is cached in the
  operating system user's credential storage or .netrc file. Subsequent commands will
  authenticate using the cached credentials. To switch users, login again using new credentials.
  To erase credentials, use the 'logout' command.

  Examples:

  - conjur login -i alice -p My$ecretPass
  - conjur login

  USAGE

    conjur login [--flags]

  FLAGS

    -d --debug     Debug logging enabled
    -h --help      Help for login
    -i --id        The identity to authenticate with. For hosts: 'host/<full path>'.
    -p --password  Password or API key for the specified identity.
    --timeout      Http timeout duration, between 1s and 10m (1m0s)
