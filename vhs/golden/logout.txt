> conjur logout --help

  Log out the user and delete the credentials cached in the operating system user's credential
  storage or .netrc file.

  USAGE

    conjur logout [--flags]

  FLAGS

    -d --debug  Debug logging enabled
    -h --help   Help for logout
    --timeout   Http timeout duration, between 1s and 10m (1m0s)
