> conjur authenticate --help

  Obtain an access token for the currently logged-in user

  USAGE

    conjur authenticate [--flags]

  FLAGS

    -d --debug   Debug logging enabled
    -H --header  Base64 encode the result and format as an HTTP Authorization header
    -h --help    Help for authenticate
    --timeout    Http timeout duration, between 1s and 10m (1m0s)
