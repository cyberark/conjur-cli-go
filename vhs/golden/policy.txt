> conjur policy --help

  Manage Secrets Manager policies

  USAGE

    conjur policy [command] [--flags]

  COMMANDS

    fetch [--flags]    Fetch effective policy
    load [--flags]     Load a policy and create resources
    replace [--flags]  Fully replace an existing policy
    update [--flags]   Update existing resources in the policy or create new resources

  FLAGS

    -b --branch        (Required) The parent policy branch
    -d --debug         Debug logging enabled
    -h --help          Help for policy
    --timeout          Http timeout duration, between 1s and 10m (1m0s)
